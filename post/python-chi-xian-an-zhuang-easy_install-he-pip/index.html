python 离线安装easy_install和pip

2019-06-13 pip ， easy_install ， python
  1. 先安装easy_install 下载源码包:https://pypi.python.org/pypi/ez_setup 解压，安装
$ python ez_setup.py
  1. 再安装pip 下载源码包:https://pypi.python.org/pypi/pip 解压，安装
$ python setup.py install
